Wedtect Project

AI-Powered Pipeline Defect Detection System

Industrial Defect Detection

Wedtect is a production-ready YOLOv8 OBB (Oriented Bounding Box) system for detecting pipeline defects including cracks, dents, holes, and leaks. Trained on 1,224 annotated images with comprehensive data augmentation and optimization.

The system provides automatic severity classification (Good/Bad/Urgent) with real-time inference capabilities suitable for industrial deployment. Includes multi-stage pipeline for data preparation, training, evaluation, and web scraping for continuous dataset expansion.
